How to Use the Diff Checker

Paste your original text in the left field and the modified text in the right field. Click "Compare" to see the differences highlighted:

  • ■ Red/removed lines — lines that exist in the original but not in the modified text
  • ■ Green/added lines — lines that exist in the modified text but not in the original
  • ■ Unchanged lines — lines that are the same in both texts

Useful for reviewing code changes, comparing document revisions, checking configuration file diffs, and verifying edits before committing.
